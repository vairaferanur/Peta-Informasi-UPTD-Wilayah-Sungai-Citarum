# Peta-Informasi-UPTD-Wilayah-Sungai-Citarum
Peta informasi Unit Pelaksana Teknis Daerah (UPTD) wilayah sungai Citarum

## Deskripsi
Repository ini hasil kegiatan magang yang dilaksanakan di bawah koordinasi Dinas Komunikasi dan Informatika Provinsi Jawa Barat, dengan penugasan di Dinas Sumber Daya Air Provinsi Jawa Barat. Peta informasi UPTD Wilayah Sungai Citarum menampilkan cakupan wilayah kerja UPTD serta informasi umum wilayah sungai. Informasi yang disajikan meliputi batas administrasi, batas wilayah sungai, batas daerah aliran sungai, dan elemen pendukung lainnya sebagai bahan visualisasi dan penyajian data wilayah sungai.

## Kegiatan
- Pengolahan dan penyajian data spasial
- Penyusunan peta informasi wilayah

## Tools
QGIS

## Output
Peta Informasi UPTD Wilayah Sungai Citarum
